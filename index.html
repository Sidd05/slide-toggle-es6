Slide Toggle widget

Demo 1 - demonstrates the following:
  • callbacks - using callbacks to add and remove a css class on button
  • clickOutside - the overlay closes by clicking anywhere on the body outside of the overlay.
  • data-clickel attribute - if you inspect the overlay, you will notice that the button is associated with the widget using the attribute
